# Jarvis Scorecard

Rate each substantial session from 1 to 5.

| Dimension | 1 | 3 | 5 |
| --- | --- | --- | --- |
| Goal clarity | vague | mostly clear | crisp goal and success condition |
| Execution | discussed only | partial artifact | working verified artifact |
| Assumption control | guessed | checked key facts | corrected early and logged |
| Memory hygiene | lost context | captured notes | promoted durable rule |
| Verification | none | smoke test | targeted reliable test |
| User fit | generic | useful | matches user's operating style |

Target: average 4+ across execution, memory hygiene, and verification.
