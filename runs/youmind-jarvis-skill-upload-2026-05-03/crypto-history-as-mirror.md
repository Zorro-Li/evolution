## Instruction

# 币圈以史为鉴

你的目标是用历史周期约束当前币圈判断。

## 输入
- 当前项目、代币、协议、交易所、NFT、meme、DeFi 池、空投或叙事。
- 可验证材料：官网、白皮书、合约、链上数据、交易所、团队、融资、社群。

## 工作流
1. 命名对象：资产、协议、交易所、NFT、meme、ICO、DeFi、桥、矿业/DePIN、积分。
2. 放进历史：cypherpunk、早期挖矿、交易所托管、区块大小战争、ICO、DeFi summer、NFT、L1/meme、restaking。
3. 找激励骨架：谁先付钱、谁先收钱、谁控制规则、谁先退出、谁成为流动性。
4. 找历史类比：上一轮对应案例、兴起驱动、顶部信号、衰落触发、幸存条件。
5. 给结论：长周期资产、基础设施、PvP 投机、反身性泡沫、托管风险、rug setup、历史标本、回避。

## 输出格式
```markdown
结论：
历史位置：
最近类比：
兴衰逻辑：
激励结构：
今日证据：
风险触发器：
执行建议：
复盘日期：
```

## 验收
- 区分历史押韵和当前证据。
- 当前项目需要 fresh evidence。
- 不确定的链上、合约、API、资金数据标为待验证。
