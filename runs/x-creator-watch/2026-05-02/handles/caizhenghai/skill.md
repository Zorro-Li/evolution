# Skill Distillation: `@caizhenghai`

## Skill Name

`x-creator-caizhenghai`

## Purpose

Track forecho, U.S. stock investor workflows, financial-resource navigation, personal performance tracking, Codex spreadsheet use cases, developer infra costs, and practical operator signals.

## Trigger Conditions

Use this skill when the user asks about:

- forecho or `@caizhenghai`
- U.S. stock X accounts and investor workflows
- brokerage/resource promotions and U.S. stock navigation pages
- personal investor performance tracking
- Codex for spreadsheets, Excel, and non-programmer work
- developer infra-cost observations tied to small operators

## Workflow

1. Capture the current visible `@caizhenghai` profile batch through Safari / Computer Use.
2. Classify each item:
   - promotion or affiliate/resource distribution
   - market observation
   - ticker mention
   - personal performance log
   - investor tool or navigation resource
   - Codex productivity workflow
   - developer/operator cost note
3. Extract:
   - ticker or product named
   - claim type
   - evidence type: screenshot, link, text, metrics
   - financial-risk level
   - Jarvis workflow implication
4. Convert the signal into local action:
   - add tickers to a watch list with verification required
   - keep promotions separate from analysis
   - turn spreadsheet examples into Codex file-workflow recipes
   - add infra-cost notes to builder-ops monitoring
   - preserve investor tracking formats for future dashboards
5. Write a brief that separates source evidence, workflow value, and financial caution.

## Failure Modes

- Treating promotions as market analysis.
- Using ticker mentions as investment advice.
- Missing practical operator details in infra-cost and spreadsheet posts.
- Ignoring the programmer side of the account.
