# Skill Distillation: `@dotey`

## Skill Name

`x-creator-dotey`

## Purpose

Track 宝玉, prompt engineering, Chinese AI/software-engineering knowledge, Codex workflows, engineering-management ideas, prompt libraries, productivity self-audits, and practical AI tool judgment.

## Trigger Conditions

Use this skill when the user asks about:

- prompt engineering and prompt libraries
- Chinese AI/software-engineering knowledge curation
- Codex workflow prompts, `/goal`, and local productivity analysis
- Chronicle or work-habit self-audit prompts
- engineering-management and AI adoption patterns
- practical cost-benefit evaluation of AI workflows

## Workflow

1. Capture the current visible `@dotey` profile batch through Safari / Computer Use.
2. Classify each item:
   - prompt library
   - Codex workflow prompt
   - local telemetry or productivity self-audit
   - AI knowledge transformation
   - engineering-management or software-engineering idea
   - local model / cost-benefit judgment
   - long-running agent signal
3. Extract:
   - exact prompt or workflow
   - source account and quoted account
   - evidence type: thread, quote, article, video, screenshot
   - sensitivity level, especially for local telemetry
   - local Jarvis architecture implication
4. Convert the signal into local action:
   - save prompt templates
   - add Codex workflow patterns to Jarvis tests
   - mark sensitive prompts that require user approval before execution
   - turn card/tutorial formats into content templates
   - map `/goal` and long-running examples to task persistence notes
5. Write a brief with prompts, proof, sensitivity flags, and recommended local experiments.

- Saving a prompt without the source context or output format.

- Amplifying local-model claims without cost-benefit judgment.
- Missing quoted source accounts that reveal early Codex workflows.
