1. Capture Daniel's profile and Blackstar link state.

3. Extract exact product nouns: Blackstar, OS, hardware, device, model training, SF, Shenzhen, AbstractVC, Naval, SV Angel, Chapter One, Timeless.

5. Map each signal to Jarvis interface strategy: desktop agent, mobile agent, wearable/device surface, local OS integration, or hardware-aware workflow.
