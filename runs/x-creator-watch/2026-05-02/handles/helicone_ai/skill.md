# x-creator-helicone

Use this skill to track `@helicone_ai`, Helicone, AI Gateway, LLM observability, open-source LLMOps, centralized billing, model routing, prompt caching, Mintlify maintenance-mode status, and background coding-agent workflow signals.

## Capture Workflow

1. Open `https://x.com/helicone_ai` in Safari through Computer Use.
2. Capture profile positioning, product status, linked website, and follower/post counts.
3. Classify each post as `gateway_positioning`, `customer_testimonial`, `cost_control`, `observability`, `company_status`, `coding_agent_signal`, or `ecosystem_retweet`.
4. Preserve source URL, metrics, quoted account, product names, and any operational status update.
5. Convert the signal into a Jarvis infrastructure implication.

## Extraction Fields

- Gateway surface: one API key, 100+ models, model routing, billing, prompt caching.
- Observability surface: analytics, traces, model usage, cost visibility.
- Product status: open-source availability, Mintlify transition, maintenance mode, platform access.
- Adjacent agent signals: Replicas, background coding agent, local codebase execution, PR reviews, Linear, Slack, API.
- Ecosystem graph: LiteLLM, Portkey, OpenRouter, Mintlify, Replicas, founder/operator accounts.

## Output Format

Return:

| Field | Content |
|---|---|
| Source | post URL and capture surface |
| Infrastructure claim | gateway, observability, cost, or product-status claim |
| Evidence | visible text, metric, quote, or linked product |
| Jarvis implication | local gateway, logging, cost-control, or coding-agent design impact |
| Verification test | small local experiment or watch query |
| Watch graph | accounts, tools, and products to monitor next |
