# Skill Distillation: @rileybrown

## Purpose

Use `@rileybrown` as a practical Codex education source for super-app framing, beginner onboarding, multitasking, skills, automations, and cross-tool AI builder workflows.

## Workflow

1. Capture profile, pinned tutorials, YouTube cards, quoted OpenAI-adjacent accounts, and high-bookmark Codex videos.
2. Classify posts by beginner onboarding, Codex feature overview, skills/plugins, automations, MCP, multitasking, app build, design, video, deck, deployment, or social automation.
3. Extract exact tool stack and workflow sequence, especially Supabase, Tally, Remotion, Canva, Vercel, Typefully, TestFlight, custom APIs, and scheduled tasks.
4. Convert each tutorial into Jarvis execution recipes: project setup, chat organization, skill creation, automation creation, artifact generation, and deployment.
5. Record engagement metrics because bookmarks identify reusable education assets.


## Failure Modes

- Treating tutorial reach as product proof.
- Losing the sequence across multiple Codex chats and artifacts.
- Capturing tool names without preserving how Riley connects them into one workflow.
