# Skill Distillation: @adocomplete

## Purpose

Use `@adocomplete` as a source for Claude Code education, Anthropic community updates, developer-tool postmortems, advanced agent patterns, and product-craft commentary.

## Workflow

1. Capture profile, pinned Claude Code resources, Anthropic/ClaudeDevs quotes, postmortems, and high-bookmark educational posts.
2. Classify posts by onboarding, advanced Claude Code capability, community announcement, quality incident, postmortem, version fix, usage-limit reset, or product taste.
3. Extract learning-sequence structure: startup basics, shortcuts, advanced capabilities, agent patterns, resources, and contextual notes.
4. Convert resources into Jarvis/skill documentation patterns with beginner-to-advanced ordering.
5. Track official ClaudeDevs quotes as source-of-truth links for reliability and incident status.

## Output Template

```markdown
## Ado / Claude Code Note

- Source:
- Resource or incident:
- Claude Code capability:
- Learning stage:
- Official source:
- Jarvis reuse:
```

## Failure Modes

- Treating community commentary as official Anthropic policy.
- Missing the source account when Ado quote-tweets ClaudeDevs.
- Capturing tips without preserving their learning progression.
