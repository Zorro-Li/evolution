# x-creator-tnm

Use this skill to track `@tnm`, Ted Nyman, OpenAI devtools, future developer tooling, Git internals, Git performance, Git recovery/repair, cloud sandboxes, EC2-as-remote-sandbox mental models, and developer-infrastructure operator signals.

## Capture Workflow

1. Open `https://x.com/tnm` in Safari through Computer Use.
2. Capture profile background, pinned/current posts, hiring posts, technical links, Git resources, cloud/runtime metaphors, and high-bookmark developer references.
3. Classify posts as `openai_devtools`, `hiring_signal`, `runtime_mental_model`, `git_internals`, `git_recovery`, `git_performance`, `developer_education`, or `low_signal_personal`.
4. Preserve source URL, visible metrics, link targets, quoted accounts, exact metaphors, and bookmark counts.
5. Convert each signal into a Jarvis developer-experience, Git reliability, or remote-runtime implication.

## Extraction Fields

- OpenAI devtools: hiring, future of devtools, team language, DM channel, workflow hints.
- Runtime model: EC2, remote sandbox, persistent workspace, cloud execution, agent runtime.
- Git expertise: `gitperf.com`, internals, performance, recovery, repair, chapter links.
- Operator background: OpenAI, Cased, GitHub CTO, BankSimple, developer tooling companies.
- Practical value: bookmark-heavy posts, free resources, exact phrases, reusable mental models.
