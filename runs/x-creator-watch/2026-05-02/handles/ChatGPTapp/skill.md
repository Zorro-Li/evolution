# Skill Draft: AI Product Culture Watcher

Use this skill when tracking consumer-facing AI product accounts such as `@ChatGPTapp`, official app accounts, AI image/video product feeds, or product-led AI meme loops.

## Inputs

- X handle
- visible posts and retweets
- quoted posts
- prompt text
- user-coined names
- linked app or account surfaces

## Workflow

1. Capture the first visible product-culture batch from Safari / Computer Use.
2. Separate native product posts from leadership retweets and user quote-tweets.
3. Normalize each item into:
   - product capability
   - prompt format
   - social label or meme name
   - source of the trend
   - engagement signal
4. Identify the repeatable behavior:
   - what users can copy
   - what name makes it spread
   - what product surface it promotes
5. Create a reusable recipe for the user:
   - prompt template
   - example output style
   - posting angle
   - follow-up naming loop

## Output Template

```md
# AI Product Culture Brief

## Product Capability
<capability>

## Viral Format
<prompt or user behavior>

## Social Label
<name users are adopting>

## Repeatable Recipe
1. <step>
2. <step>
3. <step>

## Account Signal
<what this says about the product push>
```

## Failure Modes

- Treating memes as low-signal noise.
- Copying the prompt without tracking the product behavior it exposes.
- Missing user quote-tweets that name the format.
- Mixing leadership framing with the app account’s native voice.
