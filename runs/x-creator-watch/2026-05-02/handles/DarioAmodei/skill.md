# X Creator: Dario Amodei

## Core Use

Use this skill to turn `@DarioAmodei` posts into Anthropic governance intelligence: frontier-AI risk doctrine, productized mitigation, public-policy posture, canonical essays, and rare adoption metrics.

## Inputs

- `@DarioAmodei` posts, replies, retweets, link cards, and profile metadata
- Anthropic posts retweeted by Dario
- linked essays on `darioamodei.com`
- visible engagement metrics and adoption claims
- related Anthropic programs: Project Glasswing, Claude Mythos Preview, Claude Code, interpretability work, national-security statements

## Workflow

1. Capture:
   - URL
   - timestamp label
   - text
   - whether it is original, reply-thread post, retweet, or essay link
   - linked essay/program/model
   - visible engagement metrics
2. Classify:
   - risk doctrine
   - productized mitigation
   - controlled access / release policy
   - national security / government use
   - export controls / geopolitics
   - interpretability / model understanding
   - positive AI future
   - adoption / expansion metric
3. Extract:
   - risk domain
   - mitigation mechanism
   - affected institutions
   - Anthropic product/model/program
   - policy implication
   - measurable adoption signal
4. Cross-link:
   - pair X posts with `darioamodei.com` essays
   - tie retweeted Anthropic statements back to Dario's public posture
   - add partner entities to the watch graph
5. Write a governance brief with "risk thesis", "mitigation claim", "release/access policy", "institutional partners", and "Jarvis implication".

## Pattern Library

| Pattern | What To Look For | How To Use It |
|---|---|---|
| Cyber as first frontier-risk | Project Glasswing, Mythos Preview, software vulnerability language. | Track model cyber capability and defensive deployment together. |
| Controlled model access | early access for defenders, delayed general availability, proliferation concerns. | Convert launch posts into release-governance notes. |
| Essay doctrine | Dario personal-site links and thread context. | Treat essays as canonical source documents and summarize separately. |
| National-security posture | Department of War, government use, export controls, China/DeepSeek. | Split policy analysis into state use, export controls, and democratic safeguards. |
| Interpretability urgency | model understanding, mechanistic interpretability, safety research. | Add to AI safety research monitor and future skill prompts. |
| Global adoption metric | geography, sector, Claude Code usage growth. | Preserve numeric claims and verify with follow-up Anthropic posts. |

## Failure Modes

- Treating CEO governance posts as ordinary product marketing.
- Separating model launch from release-policy and access-control decisions.
- Losing linked essays as the canonical long-form source.
- Ignoring retweeted Anthropic statements that carry Dario's public policy posture.
- Reusing adoption metrics without preserving geography, product, and time window.
