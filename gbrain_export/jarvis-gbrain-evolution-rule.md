# Jarvis GBrain Evolution Rule

User instruction: Codex should join the Jarvis evolution loop.

Operational rule:

- Before substantial tasks, run `gbrain search` for relevant local context.
- After conversations that produce reusable research plans, retrospectives, skill distillation, or important decisions, save the artifact with `gbrain put`.
- Store durable behavior changes through `python3 jarvis.py feedback` and `python3 jarvis.py promote`.
- Use GBrain as the searchable knowledge layer and Jarvis memory as the operating-rule layer.
