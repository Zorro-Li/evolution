## Durable Rule

1. Agentic infrastructure: apps, agents, deployment, Vercel AI SDK, v0, AI Gateway, frontend cloud.
2. Production proof: Ship talks, customer case studies, workshops, AI Accelerator winners.
3. Vertical-agent demand: finance, application security, real estate, and future accelerator sectors.
4. Platform security: OAuth, Google Workspace admin checks, environment variables, MFA, npm package integrity.
5. Ecosystem graph: AWS, OpenAI, Anthropic, GitHub, Microsoft, npm, SocketSecurity.
